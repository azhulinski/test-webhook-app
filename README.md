<h2>Application for testing webhook</h2>

To run App locally you have to fill `.env` file with values - for DB connection and sending mails (app uses node mail sender)

Install dependencies
### `npm install`

Run App locally
### `npm start`

To make webhook work locally you have to create a tunnel with service like ngrok and make changes in `App Script` in line `22`

Swagger documentation is awailable [here](https://test-webhook-app.onrender.com/api-docs)
